# 隐私权政策

**tt智能助手 Chrome 扩展隐私权政策**

最后更新日期：2026年1月30日

---

## 1. 引言

欢迎使用 tt智能助手（以下简称"本扩展"）。本隐私权政策旨在向您说明我们如何收集、使用、存储和保护您的信息。使用本扩展即表示您同意本隐私权政策中描述的数据处理方式。

---

## 2. 我们收集的信息

### 2.1 自动收集的信息

本扩展在您浏览支持的招聘平台时，会自动采集以下候选人信息：

| 数据类型 | 说明 |
|----------|------|
| 候选人基本信息 | 姓名、职位、公司、教育背景等 |
| 联系方式 | 手机号、邮箱（仅当招聘平台展示时） |
| 工作经历 | 工作履历、项目经验等 |
| 简历附件 | 候选人上传的简历文件 |

### 2.2 支持的平台

本扩展仅在以下招聘平台上激活并采集数据：

- 脉脉 (maimai.cn)
- BOSS直聘 (zhipin.com)
- 猎聘 (liepin.com)
- 领英 (linkedin.com)

### 2.3 用户账户信息

为了使用本扩展的完整功能，您需要登录人才管理系统账户。我们会存储：

- 用户登录凭证（Token）
- 用户标识（User ID）

### 2.4 我们不收集的信息

- 我们**不会**收集您在非招聘平台上的浏览历史
- 我们**不会**收集您的个人身份信息（除非您主动提供）
- 我们**不会**收集您的支付信息或财务数据
- 我们**不会**追踪您的地理位置

---

## 3. 信息的使用方式

我们收集的信息仅用于以下目的：

| 用途 | 说明 |
|------|------|
| 人才管理 | 将候选人信息同步到您的人才管理系统 |
| 功能提供 | 在侧边栏中展示候选人信息，提供数据管理功能 |
| 服务改进 | 分析使用情况以改进产品功能（匿名数据） |
| 技术支持 | 诊断和修复技术问题 |

---

## 4. 信息的存储和安全

### 4.1 本地存储

本扩展使用浏览器的本地存储（localStorage 和 chrome.storage）来保存：

- 用户登录状态
- 扩展配置信息
- 临时缓存数据

### 4.2 服务器传输

采集的候选人数据会通过 HTTPS 加密传输到人才管理系统服务器。

### 4.3 安全措施

我们采取以下措施保护您的数据安全：

- 所有数据传输均使用 HTTPS 加密
- 服务器采用行业标准的安全防护措施
- 定期进行安全审计和漏洞修复
- 访问控制和权限管理

---

## 5. 信息共享

我们**不会**将您的信息出售、出租或以其他方式提供给第三方，除非：

- 获得您的明确同意
- 法律法规要求披露
- 保护我们的合法权益

---

## 6. 数据保留

- 本地缓存数据会在您卸载扩展时自动删除
- 服务器上的数据按照人才管理系统的数据保留政策进行管理
- 您可以随时请求删除您的账户和相关数据

---

## 7. 您的权利

根据适用的数据保护法律，您享有以下权利：

| 权利 | 说明 |
|------|------|
| 访问权 | 您有权访问我们收集的关于您的数据 |
| 更正权 | 您有权更正不准确的个人数据 |
| 删除权 | 您有权要求删除您的个人数据 |
| 限制处理权 | 您有权限制我们处理您的数据 |
| 数据可携权 | 您有权以通用格式获取您的数据 |
| 撤回同意权 | 您有权随时撤回您的同意 |

如需行使上述权利，请通过下方联系方式与我们联系。

---

## 8. Cookie 和类似技术

本扩展**不使用** Cookie 进行用户追踪。我们仅使用浏览器提供的本地存储 API 来保存必要的配置和状态信息。

---

## 9. 儿童隐私

本扩展不面向 13 岁以下的儿童。我们不会故意收集儿童的个人信息。如果您发现我们无意中收集了儿童的信息，请立即联系我们，我们将采取措施删除相关数据。

---

## 10. 隐私政策的变更

我们可能会不时更新本隐私权政策。更新后的政策将在本页面发布，并注明最后更新日期。重大变更时，我们会通过扩展内通知或其他适当方式告知您。

建议您定期查看本隐私权政策，以了解我们如何保护您的信息。

---

## 11. 权限说明

本扩展请求以下浏览器权限：

| 权限 | 用途说明 |
|------|---------|
| `host_permissions` | 访问招聘平台网站以采集候选人信息 |
| `scripting` | 在招聘网站注入脚本以提取数据 |
| `activeTab` | 获取当前标签页信息 |
| `tabs` | 管理标签页状态 |
| `storage` | 存储用户配置和登录状态 |
| `webNavigation` | 监听页面导航事件 |
| `webRequest` | 拦截网络请求获取候选人详情 |
| `sidePanel` | 提供侧边栏界面 |
