# 隐私政策 (Privacy Policy)

**生效日期：** 2024年1月16日

## 1. 简介
本隐私政策描述了 **Agent 调试工作台** ("我们", "本扩展") 如何收集、使用和披露您的信息。我们致力于保护您的隐私，并确保您了解我们如何处理数据。

## 2. 我们收集的信息
本扩展 **不收集** 任何个人身份信息 (PII)，如您的姓名、地址或电话号码，除非您在使用特定功能（如发布或保存 Prompt）时主动通过 API 交互提供。

### 2.1 自动收集的信息
- **设备信息：** 我们不收集设备指纹或唯一设备标识符。
- **扩展使用数据：** 所有的 Prompt 数据、评测记录和配置信息均存储在您的本地浏览器中 (`chrome.storage.local`)，或直接发送至您配置的企业内部网关 API。

### 2.2 用户主动提供的信息
- **Prompt 和配置：** 您在扩展中编写的 Prompt 和配置数据。
- **用户信息：** 当您在宿主页面登录时，扩展可能会读取当前用户的 ID（如 `unique_id`）以用于 API 请求的鉴权和记录归属。

## 3. 信息的使用
我们收集的信息仅用于以下目的：
- 提供扩展的核心功能，如 Prompt 调试、版本管理和效果评测。
- 通过 API 将您的配置同步到指定的服务器（仅在您点击保存或发布时）。
- 改善扩展的用户体验。

## 4. 数据存储与安全
- **本地存储：** 您的偏好设置和缓存数据存储在您浏览器的本地存储中。
- **数据传输：** 与后端服务器的所有通信均通过 HTTPS 加密传输。我们不与任何第三方广告商或数据代理商共享您的数据。

## 5. 第三方服务
本扩展可能会与您配置的内部网关或第三方 API（如大模型服务）进行交互。这些交互受各自服务提供商的隐私政策约束。

## 6. 您的权利
您有权随时卸载本扩展，卸载后，扩展在本地存储的所有数据将被清除（浏览器同步数据除外）。

## 7. 变更
我们需要时可能会更新本隐私政策。任何更改都将在本页面上发布。
